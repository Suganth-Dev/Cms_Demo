----------------------------------------
JSX MIME TYPE NOTE
----------------------------------------

This site uses in-browser Babel to compile JSX at runtime.
JSX files (.jsx) must be served by the web server.

If BigRock returns a 404 or refuses to serve .jsx files:
- Check that .jsx is not blocked in the server configuration.
- Add the following to .htaccess in the web root if needed:

  AddType application/javascript .jsx

If the server uses nginx, add to the server block:
  types { application/javascript jsx; }

BigRock cPanel hosting typically allows .htaccess overrides.
